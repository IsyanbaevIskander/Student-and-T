# Проект "Студент и Т"
Данный проект реализован в рамках всероссийского хакатона "Чемпионат Урала".
## Идея проекта 
Бронирование рабочих мест в IT-хабах Т-Банка.
Есть три типа пользователей: 
1) обыкновенный пользователь, который может бронировать места, подавать заявку на ментора
2) ментор (работник Т-Банка). Он может принимать заявки от пользователей, чтобы проконсультировать по вопросам.
3) Администратор. Управление IT-хабами, мониторинг статистики, принятие заявок на проведение мероприятий.
## Стек технологий
Backend: FastAPI + SQLAlchemy.
Frontend: React
